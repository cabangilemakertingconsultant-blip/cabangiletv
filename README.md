# 📺 Cabangile TV PRO

Welcome to **Cabangile TV PRO**, a web-based digital TV platform designed to stream YouTube playlists in a clean, TV-style interface.

## 🌟 Features

- 📺 TV-style video player
- 🎬 Multiple YouTube playlist support
- ▶️ One-click playlist switching
- 📱 Responsive design for desktop and mobile
- 📡 Live TV interface
- 💰 Google AdSense ready layout
- ⚡ Fast, lightweight, and easy to host on GitHub Pages

## 🚀 Getting Started

1. Download or clone this repository.
2. Open `index.html` in your web browser.
3. Or upload the project to GitHub Pages.

No installation is required.

## 🌐 Deploy on GitHub Pages

1. Create a new GitHub repository.
2. Upload all project files.
3. Go to **Settings** → **Pages**.
4. Select the **main** branch.
5. Save the settings.
6. Your Cabangile TV website will be published online.
